# RAFIK FOCUS

تطبيق تنظيم الوقت وتنفيذ خطة العمل ديال RAFIK STOORE.

## كيفاش تنشرو (Vercel — مجاني)

1. دير حساب على Vercel بحساب GitHub ديالك
2. دوس "Add New Project" واختار هاد المستودع
3. Vercel غادي يعرف وحدو أنه مشروع Vite/React، غير دوس "Deploy"
4. غادي يعطيك رابط بحال rafik-focus.vercel.app

## كيفاش تخليه بحال تطبيق فالهاتف

1. حل الرابط ديالك فـ Chrome
2. ⋮ → "Add to Home screen"
